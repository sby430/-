# 医院门诊管理数据库系统

## 项目简介
本项目为数据库系统课程大作业，基于 MySQL 设计并实现一个社区医院门诊管理数据库，
支持患者预约、到院登记、就诊记录、缴费结算及统计分析等功能。

## 技术环境
- 数据库：MySQL 8.0
- 管理工具：Navicat
- 操作系统：Windows 11

## 功能模块
- 患者信息管理
- 医生与科室管理
- 预约挂号
- 就诊记录管理
- 缴费结算
- 门诊收入统计

## 使用说明
1. 在 MySQL 中执行 create_tables.sql 创建表结构
2. 执行 insert_test_data.sql 插入测试数据
3. 执行 create_admin.sql 创建管理员
4. 执行 test_for_video 视频中实现测试
